# Referent

Минимальное Next.js приложение.

## Установка

```powershell
npm install
```

## Запуск в режиме разработки

```powershell
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для продакшена

```powershell
npm run build
npm start
```

## Технологии

- Next.js 14
- React 18
- TypeScript
